# Bininfo 
Get information about various binary file formats 

Warning: Error handling is very basic and hasn't been tested.

![png_example1](Media/Screenshot1.png)

## Supported formats
- [x] BMP
- [x] PNG 1.2, PNGEXT 1.2
- [x] GIF
- [ ] JPG
- [ ] PDF
- [WIP] ELF
- [ ] PE
